# Changelog

All notable changes to this project are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-17

### Added
- Initial repository bootstrap with `main` default branch and baseline structure.
- FinOps planning documentation with nomenclature, objectives, implementation phases, and DoD.
- Public roadmap via GitHub milestones and task issues.
- Governance baseline:
  - `CONTRIBUTING.md`
  - `SECURITY.md`
  - `CODEOWNERS`
  - PR and issue templates
  - label taxonomy
- CI/security automation:
  - quality workflow
  - dependency review workflow
  - CodeQL workflow
  - Dependabot configuration

### Changed
- Branch protection hardened with required checks and linear history.
- Merge strategy set to squash-only with automatic branch deletion on merge.
